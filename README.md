# Student Mark Prediction

**B.Tech 2nd Year AI&DS | MSEC**

## About
This project predicts student marks using Python, Pandas, Matplotlib and Scikit-learn.

## Tech Stack
- Python
- Jupyter Notebook
- Pandas, Matplotlib, Scikit-learn

## How to Run
1. Install libraries:
   pip install pandas matplotlib scikit-learn
2. Run the notebook:
   student_mark_prediction.ipynb
